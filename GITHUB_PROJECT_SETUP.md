# GitHub Project Setup Guide

## Recommended Project Name

SecAgent RiskOps Roadmap

## Recommended Views

### 1. Roadmap View

Use this view for long-term planning.

Fields:
- Phase
- Target Version
- Status
- Priority
- Type

### 2. Board View

Use this view for daily development.

Columns:
- Backlog
- Ready
- In Progress
- Blocked
- Review
- Done

### 3. Current Sprint

Filter:
- Status is not Done
- Priority is P0 or P1

### 4. Architecture / Docs

Filter:
- Type is Architecture or Documentation

2. Create labels.
3. Create milestones.
4. Create GitHub Project.

6. Create initial issues.
7. Add issues to project.
8. Start with v0.1 Foundation and v0.2 AI SOC Inbox.
