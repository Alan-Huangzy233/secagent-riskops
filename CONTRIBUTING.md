# Contributing

## Development Principles

1. Evidence-grounded outputs only.
2. Deterministic detection before AI interpretation.
3. AI proposes, policy decides, executor acts.
4. All remediation must support verification.
5. High-risk actions require approval.
6. Knowledge promotion requires validation.

## Issue Types

- Architecture
- Backend
- Frontend
- Agent
- SOC
- GRC
- Executor
- Security
- Testing
- Documentation
- DevOps

## Pull Request Checklist

- [ ] Tests added or updated
- [ ] Documentation updated
- [ ] Security impact considered
- [ ] Evidence and audit implications considered
- [ ] No unrestricted command execution added
- [ ] No secrets committed
