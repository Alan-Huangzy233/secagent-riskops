# Autonomy Levels

| Level | Name | Description |
|---|---|---|
| 0 | Report Only | Analyze and report only. |
| 1 | Suggest Only | Suggest remediation but do not change systems. |
| 2 | Create Patch / PR | Generate code patches or pull requests only. |
| 3 | Execute After Approval | Execute actions only after approval. |
| 4 | Auto-Fix Low Risk | Automatically fix low-risk issues; require approval for higher risk. |
| 5 | Lab-Only Full Autonomy | Full autonomy only in controlled lab environments. |

Default recommended mode: Level 2 or Level 3.
